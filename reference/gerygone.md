# Lord Howe Gerygone sightings

Lord Howe Gerygone (*Gerygone insularis*) sighting record based on data
from the Atlas of Living Australia. The temporal range (`init.time` to
`test.time`) of these data is 1788 to 2023. Format: `cdis`.

## Usage

``` r
gerygone
```

## Format

a `numeric` vector with 236 values.
