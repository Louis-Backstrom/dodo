# Caribbean Monk Seal sightings

Caribbean Monk Seal (*Monachus tropicalis*) sighting record from Solow
(1993) based on data presented in LeBouef et al. (1986). The temporal
range (`init.time` to `test.time`) of these data is 1915 to 1992.
Format: `ccon`.

## Usage

``` r
monk_seal
```

## Format

A `numeric` vector with 5 values.
