# Alroy's (2014) "Creeping Shadow of a Doubt" model

The model from Alroy 2014. Estimates a posterior probability that the
species is extant at the end of the observation period.

## Usage

``` r
AL14B1(records)
```

## Arguments

- records:

  sighting records in `cbin` format (see
  [`convert_dodo`](https://louis-backstrom.github.io/dodo/reference/convert_dodo.md)
  for details).

## Value

a `list` object with the original parameters and p(extant) included as
elements.

## Note

All sighting records are assumed to be certain and sampling effort is
assumed to be constant.
