# Example sightings from Thompson et al.

Example multi-class sighting record from Table 1 in Thompson et al.
(2013). The temporal range (`init.time` to `test.time`) of these data is
0 to 7. Format: `umcd`.

## Usage

``` r
thompson_table1
```

## Format

a `data.frame` with 4 columns and 7 rows.

## References

**Key Reference**

Thompson, C. J., Lee, T. E., Stone, L., McCarthy, M. A., & Burgman, M.
A. (2013). Inferring extinction risks from sighting records. *Journal of
Theoretical Biology*, 338, 16-22.
[doi:10.1016/j.jtbi.2013.08.023](https://doi.org/10.1016/j.jtbi.2013.08.023)
