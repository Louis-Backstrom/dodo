# Example sightings from Burgman et al.

Example frequency-based sighting record from Figure 1b in Burgman et al.
(1995). The temporal range (`init.time` to `test.time`) of these data is
0 to 16. Format: `cdis`.

## Usage

``` r
burgman_figure1b
```

## Format

a `numeric` vector with 16 values.

## References

**Key Reference**

Burgman, M. A., Grimson, R. C., & Ferson, S. (1995). Inferring Threat
from Scientific Collections. Conservation Biology, 9(4), 923-928.
[doi:0.1046/j.1523-1739.1995.09040923.x](https://doi.org/0.1046/j.1523-1739.1995.09040923.x)
