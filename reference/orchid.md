# Mauritian orchid sightings

Mauritian orchid (*Angraecum* aff. *rutenbergianum*) sighting record
from Solow & Roberts (2003). The temporal range (`init.time` to
`test.time`) of these data is 1933 to 1996. Format: `ccon`.

## Usage

``` r
orchid
```

## Format

A `numeric` vector with 11 values.

## References

**Key Reference**

Solow, A. R., & Roberts, D. L. (2003). A nonparametric test for
extinction based on a sighting record. *Ecology*, 84(5), 1329-1332.
[doi:10.1890/0012-9658(2003)084\[1329:ANTFEB\]2.0.CO;2](https://doi.org/10.1890/0012-9658%282003%29084%5B1329%3AANTFEB%5D2.0.CO%3B2)
