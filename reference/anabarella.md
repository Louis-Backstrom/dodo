# *Anabarella* fossil record

Siberian and Mongolian records of Cambrian *Anabarella* sp. from Wang et
al. (2016) based on data presented in Maloof et al. (2010). Format:
`ccon`.

## Usage

``` r
anabarella
```

## Format

A `numeric` vector with 19 values.

## Note

Dates are measured in million years before present (note that values are
all \<0). Radiometric dating error estimates are not included.
