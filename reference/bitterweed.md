# Bitterweed sightings

Bitterweed (*Helenium amarum*) sighting record from Rout et al. (2009).
Format: `cbin`.

## Usage

``` r
bitterweed
```

## Format

a `numeric` vector with 178 values.

## References

**Key Reference**

Rout, T. M., Salomon, Y., & McCarthy, M. A. (2009). Using sighting
records to declare eradication of an invasive species. *Journal of
Applied Ecology*, 46(1),
110-117.[doi:10.1111/j.1365-2664.2008.01586.x](https://doi.org/10.1111/j.1365-2664.2008.01586.x)
