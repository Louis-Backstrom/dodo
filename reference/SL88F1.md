# Springer & Lilje's (1988) "Broken Stick" model

Equation 3 from Springer & Lilje 1988. Estimates a one-sided \\1 -
\alpha\\ confidence interval on the time of extinction.

## Usage

``` r
SL88F1(records, alpha = 0.05)
```

## Arguments

- records:

  sighting records in `ccon` format (see
  [`convert_dodo`](https://louis-backstrom.github.io/dodo/reference/convert_dodo.md)
  for details).

- alpha:

  desired significance level (defaults to \\\alpha = 0.05\\) of the
  \\1 - \alpha\\ confidence interval.

## Value

a `list` object with the original parameters and the confidence interval
included as elements. The confidence interval is a two-element numeric
vector called `conf.int`.

## Note

All sighting records are assumed to be certain and sampling effort is
assumed to be constant.

## Examples

``` r
# Run an example analysis using the Caribbean Monk Seal data
SL88F1(monk_seal)
#> $records
#> [1] 1915 1922 1932 1948 1952
#> 
#> $alpha
#> [1] 0.05
#> 
#> $conf.int
#> [1] 1952.000 1979.711
#> 
if (FALSE) { # \dontrun{
# Run an example analysis using the Slender-billed Curlew data
SL88F1(curlew$ccon)
} # }
```
